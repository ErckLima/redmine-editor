# Editor Redmine - Preview em Tempo Real

Ferramenta para edição de texto com formatação Redmine (Textile), com preview em tempo real.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Digite seu texto com formatação Redmine no painel esquerdo
3. Veja a formatação em tempo real no painel direito
4. Clique em **Copiar** para copiar o texto e colar no Redmine

## Formatações suportadas

- `*negrito*`, `_itálico_`, `+sublinhado+`, `-riscado-`
- `h1.`, `h2.`, `h3.` para títulos
- `* item` para listas com bullets, `# item` para listas numeradas
- `bq.` para citações
- `p=.` centralizado, `p>.` alinhado à direita
- `---` para linha horizontal
- Links, imagens, menções e muito mais
